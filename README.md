Данный проект - микросервис по созданию и управлению категориями, выполняет все CRUD-операции с ними.
Также управляет темами. Обеспечивает получение по категорям, идентификаторам.
REST API, совместимый с другими микросервисами CheckDev.

Применяемые технологии:
SpringBoot,
Liquibase,
Spring Security,
Spring Data JPA
PostgreSQL
Java 17 и выше
Maven
REST API
Jenkins
